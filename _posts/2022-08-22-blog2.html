The second experiment I explored was how to insert graphics and change the coordinates of the graphics to become mouseX, mouseY in p5js, allowing the graphics to interact with the mouse. First I set the background to red, then I made a quad and gave it black, and finally changed some of the coordinates of the quad to become mouseX, mouseY, so that the trapezoid interacted with my mouse in an interesting way.

2.Here is my code

function setup() {
createCanvas(576, 324);
textFont("华文彩云");
fill('#3F51B5')
stroke(300)
}
function draw() {
background(200,200);
var locX=random(0,400);
var locY=random(0,400);
var s=random(10,60);
square(locX,locY,s)
text("RMIT", 25, 60);
textSize(32);
text("Creative Coding Specialisation", 27, 90);
textSize(64);
}
